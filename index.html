About

Projects

Project 1: Jurassic Pac-Man

A week-long solo project to create a grid-based game to be rendered in the browser, using 'vanilla' JavaScript, HTML and CSS. I created a Jurassic Park themed version of the arcade classic Pac-Man that allows the user to control a Jeff Goldblum avatar, losing lives when caught by pursuing dinosaurs who move intelligently around the maze, and achieving a final score if they win.

Project 2: Weather API

A two-day hackathon paired project to create a React app that consumes a public API and has several components, both functional and classical. DarkSky API was our data source, and Bulma was used as a framework for our styling. We used moment.js to create a clock that ticks in local time on each city page.

Project 3: Food for Thought

A week-long project, working in a team of 4, to build a full-stack application. We used Express to create our API and serve our data from a Mongo database; this API was consumed with a React front-end. We used Git and Github to work collaboratively, developing features on our own branches before merging to a master branch. I built both front-end features, including a search and filter function for finding a restaurant that work in tandem, and back-end features, including a ‘change password’ function.

Project 4: Recipedia

A week-long paired project to build a full-stack application by making our own back-end and front-end. We created a PostgreSQL database with selected data taken from Edamam public API, served this data using Django REST Framework, and we built our front-end using React. I found it particularly satisfying designing backend models in Django to serve data in the most useful way. I built both back-end and front-end advanced search features, using Django filter queries using Python on the back-end, and developed my skills with React hooks on the front-end.
